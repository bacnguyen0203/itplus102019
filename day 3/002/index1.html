Thuộc tính style trong các thẻ html
        style=""
        Bên trong nháy kép sẽ là mã code css
        CSS là 1 ngôn ngữ trang trí cho website

        Code của css trong thẻ style=""
        nó sẽ đi theo cặp "tên_thuộc_tính_css:giá_trị"
        Trường hợp có nhiều thuộc tính css trong thuộc tính html style
        thì cú pháp như sau style="tên_thuộc_tính_css1:giá_trị_1;tên_thuộc_tính_css2:giá_trị_2 ..."
        phân tách các cặp thuộc_tính_css:giá_trị bằng dấu ;

        Làm quen với 1 số thuộc tính css cơ bản
        Có 1 điều phải nhớ : thuộc tính html và thuộc tính css thì khác nhau
        Ví dụ 1 vài thuộc tính css như sau :
        color ( màu sắc trong css ) có các giá trị như là : black,while,blue,#FFFFFF
        background-color ( màu nền trong css )
        font-size ( kích cỡ chữ to hay nhỏ trong css )
